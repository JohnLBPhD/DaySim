# DaySim

DaySim Activity-Based Model

How to Build 
-  Build with Microsoft Visual Studio Express 2015 or later
-  Open DaySim.sln solution  
-  Build + Rebuild Solution outputs DaySim_dist\DaySim.exe

How to Run
-  DaySim.exe -c configuration.xml
